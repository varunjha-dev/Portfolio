# **Portfolio Website**

A responsive and interactive portfolio website showcasing projects, skills, and contact information. Built using **HTML** and **CSS** 

---

## **Features**

- 🌟 **Responsive Design**: Adapts seamlessly to all devices and screen sizes.
- 🔗 **Interactive Navbar**: Hamburger menu for smaller screens with smooth transitions.
- 🎨 **Portfolio Showcase**: Highlighted projects with interactive cards.
- 🚀 **Skills Section**: Visual representation of technical skills using progress bars.
- 📜 **Smooth Scrolling**: Enhances user navigation experience.
- 🔝 **Back-to-Top Button**: Allows easy navigation to the top of the page.
- 🎬 **Dynamic Animations**: Scroll-based animations for portfolio and skills sections.

---

## **Tech Stack**

- **HTML5**: Semantic structure for easy navigation and accessibility.
- **CSS3**: Styling with variables, animations, and media queries for responsiveness.
- **JS**: for the purpose of navbar toggle, scroll behavior etc.,
